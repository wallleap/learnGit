Git 学习

1、初始化环境

 下载 Git、注册 GitHub 等账号

 配置本地 Git 信息

  • git config --global user.name "Your Name"
  • git config --global user.email "email@example.com"
  •         # 查看配置信息（按 q 退出）
        git config --list

添加了 --global 是针对全局生效的，如果某个仓库需要单独的，只需在这个仓库中使用没有 --global 的命令配置一遍即可

 生成 keygen

ssh-keygen -t rsa -C "email@example.com"

或使用 ed25519 算法

ssh-keygen -t ed25519 -C "SSH Key"

将生成的 id_rsa.pub 中的公钥放到需要提交的服务器上（~/.ssh/authorized_keys 文件中）

 测试连接

ssh -T git@github.com

2、了解Git基本命令

工作区（working tree），暂存区（index /stage），本地仓库（repository）

 本地仓库

初始化本地仓库

git init

创建 .gitignore 文件

      touch .gitignore
      # 修改这个文件，将需要忽略的文件名/目录写入
      vi .gitignore

查看仓库中文件状态

      git status

 暂存区

添加当前目录下文件到暂存区

      git add .
      # 添加部分文件
      git add 文件名|文件夹

将暂存区中的文件进行提交

      git commit -m "本次提交的说明"

 远程仓库

添加远程仓库

      git remote add origin(名字) 仓库链接(ssh或https)

查看远程仓库

      git remote -v
      # 删除某个远程连接
      git remote remove origin(名字)

推送本地代码到远程仓库

      git push origin main:main
      # origin 是远程仓库别名、前一个 main 是本地分支名、后一个 main 是远程分支名
      # 第一次可以多加一个参数 -u，将设置默认推送目标
      git push -u origin master
      # 还有其他参数，-f 可以强制推送，--all 推送所有分支，--tag 推送所有标签

拉取远程代码到本地仓库

      git pull origin main:main
      # origin 是远程仓库别名、前一个 main 是远程分支名、后一个 main 是本地分支名
      # --rebase 参数使用变基，-r 禁用变基强制使用合并
      # fetch 下载最新更改，但不自动合并到当前分支
      git fetch

克隆远程代码到本地仓库

      git clone 仓库链接 目录名
      # -b main 指定克隆分支，--depth 1 指定克隆深度

 查看文件修改内容

      git diff 文件名

 查看历史记录

      git log
      git log --pretty=oneline

 查看引用日志

      git reflog

 版本回退

      # 回退到上一个版本(Git 中 HEAD 代表当前版本)
      git reset --hard HEAD^
      # 回退到上上个版本
      git reset --hard HEAD^^
      # 回退到上100个版本
      git reset --hard HEAD~100
      # 根据版本号回退
      git reset --hard 版本号前几位或写全

 分支管理

      # 创建分支
      git branch 分支名
      # 切换分支
      git checkout 分支名
      git switch 分支名
      # 可以合写为一条
      git checkout -b 分支名
      git switch -c 分支名
      # 查看分支，-r 参数查看远程分支
      git branch
      # 合并分支到当前分支
      git merge 分支名
      # 删除分支
      git branch -d 分支名
